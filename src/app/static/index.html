6. Live Stream (RTSP / Webcam / File)

Default points at the local RTSP demo at rtsp://localhost:8554/traffic, served by bash scripts/start_rtsp.sh --detach. You can also enter a local file path, an external RTSP URL, or 0 for the default webcam. The feed is served as MJPEG — one classified frame per HTTP part — with YOLO + MobileViT.
